# Lautaro Altamirano — Portfolio

Landing profesional en Next.js enfocada en perfil **Backend Developer** orientado a:
- sistemas reales
- automatización
- SQL
- integración entre sistemas

## Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Ejecutar localmente

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Lint

```bash
npm run lint
```

## Build de producción

```bash
npm run build
npm start
```
